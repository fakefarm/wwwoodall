Craftsmanship

Feb 2015

To much reduce()

Working on a story that was missing data. I followed the path and discovered that map() was being used and removing everything but an id. It was an easy fix to simply return a hash instead of an integer. That helped me see how reduce can cut out more info than it should. Not a bug but a change in code requirements. It was the first time I had seen map, over reducing.
