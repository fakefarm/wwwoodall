Dyanmic pages

Dec 21, 2014

Just because a static site is basic doesn't mean you have to copy and paste all over the place. Middleman takes advantage of partials just like Rails does. Another awesome feature of Middleman is dynamic pages. The way these work is you make a template for common pages. For example, if you wanted to have a different page for each player on the team. Instead of managing 42 different pages, you can create one template then hold the content of each player in a yaml file.

The dynamic page will create 42 separate pages for you. It's an awesome feature. I'm using with a directory called templates which houses four different templates for me. I base them on the four primary uses of content that data flow through. Grid, Notes, Lists, and Jots.
