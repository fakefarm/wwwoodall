Upgrading rails versions

Jun 23, 2015

Today was the first time I was tasked with upgrading a Rails version. It was only a security patch but I was able to handle the process and learn a few things along the way. First, update the gemfile to whatever new rails version you're want

# GEMFILE
gem rails 4.1.11

Then, run

$ bundle update rails

You'll either be good to go or you might hit some errors that advise that you to install a different gem. For me, I hit a few errors and after each error, bundle told me to make sure a particular gem was installed before proceeding. But gem install foo wasn't solving my errors.

That's when Jon Phillips pointed out that instead of installing those gem, I should update them as well with bundle update foo. If your terminal comes back saying you need to update rails, then you should chain them together like so;

bundle update rails foo

Lastly, run that test suite and you're good to go!

Thanks, Jon!
